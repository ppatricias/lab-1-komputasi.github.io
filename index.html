Lab-1: Implementasi dan pemanfaatan infrastruktur perangkat keras menggunakan ownCloud pada Sistem operasi linux Ubuntu 18.04 server

Tujuan :

  • Mahasiswa bisa mengimplementasikan sistem operasi berbasis web interface, pemanfaatan storage berbasis cloud.

Perkenalan.

Owncloud adalah sebuah platform file sharing yang bersifat open-source dan berbagai gabungan atau kolaborasi yang dapat menyimpan, memanage personal content kita, seperti dokumen dan gambar pada lokasi yang terpusat. Implementasi dari ownCloud merupakan hal yang serupa pada aplikasi dropbox, dengan fitur online storage yang dapat kita create sesuai dengan resource pada server atau komputer kita.

Langkah 1 - Instalasi Ubuntu Server.

Pertama kali, kita harus mendownload file .iso linux ubuntu server pada link berikut https://ubuntu.com/download/server

 

Lakukan instalasi di komputer yang tersedia, ataupun jalankan menggunakan software virtualisasi seperti virtualbox (https://www.virtualbox.org/wiki/Downloads) ataupun optional menggunakan VMware player (https://www.vmware.com/go/downloadworkstationplayer). Untuk tutorial tata cara proses instalasi ubuntu server pada aplikasi virtualisasi akan dilewatkan.

*Catatan: pastikan alamat IP pada virtualbox terpasang dan dalam network yang sama.

Langkah2 - Instalasi ownCloud

Paket ownCloud server pada dasarnya tidak tersedia pada repositori dari sistem linux ubuntu tersebut, maka perlu kita unduh release key menggunakan curl command dan import menggunakan apt-key add, seperti pada command di bawah:

$ curl https://download.owncloud.org/download/repositories/10.0/Ubuntu_18.04/Release.key | sudo apt-key add -

File ‘Release.key’ berisikan sebuah PGP (Pretty Good Privacy public key seperti apt, yang akan digunakan untuk men-verifikasi ownCloud package sebagai aplikasi authentic.

Untuk tambahan silahkan tambahkan owncloud.list pada source.list.d pada direktori apt.

$ echo 'deb http://download.owncloud.org/download/repositories/10.0/Ubuntu_18.04/ /' | sudo tee /etc/apt/sources.list.d/owncloud.list

Kemudian lakukan update, instalasi beberapa paket - paket pendukung seperti php

$ sudo apt update

$ sudo apt install php-bz2 php-curl php-gd php-imagick php-intl php-mbstring php-xml php-zip owncloud-files

Langkah 3 - konfigurasi MySQL database

Login ke database sql server menggunakan command line sebagai berikut:

$ sudo mysql

Lakukan konfigurasi untuk password MySQL pada akun root:

$ mysql -u root -p

Kemudian, buatlah database untuk ownCloud:

$ mysql> CREATE DATABASE owncloud;

$ mysql> GRANT ALL ON owncloud.* to 'owncloud'@'localhost' IDENTIFIED BY 'owncloud_database_password';

*Catatan: ganti owncloud_database_password dengan password database.

$ mysql> FLUSH PRIVILEGES;

$ mysql> exit

Setelah login menggunakan akun, berikut tampilan ownCloud.

Berikut adalah penjelasan tentang beberapa fitur ownCloud beserta pemanfaatannya:

  • Penyimpanan file

OwnCloud menyediakan fitur untuk menyimpan file secara aman dan terenkripsi di server. Pemanfaatannya adalah sebagai media penyimpanan data digital dengan akses yang dapat dilakukan dari berbagai perangkat yang terhubung ke ownCloud.

  • Sinkronisasi

OwnCloud memungkinkan sinkronisasi otomatis antara perangkat pengguna dan server ownCloud. Pemanfaatannya adalah untuk memastikan bahwa file yang disimpan di server ownCloud selalu terupdate dan dapat diakses dari perangkat mana saja tanpa harus mengunggah atau mengunduh manual.

  • Kolaborasi

OwnCloud memungkinkan pengguna untuk berkolaborasi dengan pengguna lain dalam proyek yang sama. Pemanfaatannya adalah untuk meningkatkan produktivitas dan efisiensi dalam pekerjaan yang memerlukan kerja sama antar tim.

  • Favorites

Fitur ini memungkinkan pengguna untuk menandai file dan folder favorit mereka untuk memudahkan akses di kemudian hari. Pemanfaatannya adalah untuk menghemat waktu dan usaha dalam mencari file dan folder yang sering digunakan.

  • Shared with you

Fitur ini memungkinkan pengguna untuk melihat file dan folder yang dibagikan kepada mereka oleh pengguna lain. Pemanfaatannya adalah untuk memudahkan pengguna dalam mengakses file yang dibagikan dan meningkatkan kolaborasi.

  • Shared with other

Fitur ini memungkinkan pengguna untuk melihat file dan folder yang mereka bagikan dengan pengguna lain. Pemanfaatannya adalah untuk memudahkan pengguna dalam mengelola dan mengendalikan akses file dan folder yang dibagikan.

  • Shared by link

Fitur ini memungkinkan pengguna untuk membagikan file dan folder dengan membuat tautan yang dapat dibagikan kepada pengguna lain. Pemanfaatannya adalah untuk memudahkan pengguna dalam berbagi file dan folder dengan orang di luar ownCloud.

  • Tags

Fitur ini memungkinkan pengguna untuk menambahkan label atau tag pada file dan folder untuk memudahkan pengelompokan dan pencarian. Pemanfaatannya adalah untuk memudahkan pengguna dalam menemukan file dan folder yang relevan.

  • Deleted files

Fitur ini memungkinkan pengguna untuk melihat file dan folder yang telah dihapus dan memulihkannya jika diperlukan. Pemanfaatannya adalah untuk memastikan bahwa pengguna tidak kehilangan file dan folder yang penting secara permanen.

  • Settings

Fitur ini memungkinkan pengguna untuk mengatur preferensi dan pengaturan ownCloud mereka, seperti mengubah bahasa, mengaktifkan notifikasi, dan lain-lain. Pemanfaatannya adalah untuk menyesuaikan ownCloud dengan kebutuhan pengguna.

  • Search

Fitur ini memungkinkan pengguna untuk mencari file dan folder berdasarkan nama, tag, atau jenis file. Pemanfaatannya adalah untuk memudahkan pengguna dalam menemukan file dan folder yang mereka butuhkan.

  • Create

Fitur ini memungkinkan pengguna untuk membuat file dan folder baru langsung dari ownCloud. Pemanfaatannya adalah untuk memudahkan pengguna dalam membuat file dan folder baru tanpa harus keluar dari ownCloud.
